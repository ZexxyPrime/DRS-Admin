Configuration
All settings sync automatically to all team members via Firebase
✅
All config synced from Firebase
Team members get all settings automatically when they open this panel
⚠️
Enter Firebase config below to activate team sync
Once Firebase is set, all other configs sync automatically to every team member
🔥
Step 1 — Firebase Config (Master)
Get from: Firebase Console → Your Project → Project Settings → General → Your Apps → Web App → Config object.
Set Firebase Rules first: Firebase Console → Realtime Database → Rules → paste: { "rules": { ".read": true, ".write": true } } → Publish
🗄️
Step 2 — Supabase Database
Go to supabase.com → Create free project → SQL Editor → Run this SQL once:
CREATE TABLE IF NOT EXISTS leads ( id BIGSERIAL PRIMARY KEY, ref_id TEXT, full_name TEXT, email TEXT, whatsapp TEXT, alt_phone TEXT, service TEXT, sub_type TEXT, workers_required INTEGER, engagement_type TEXT, urgency TEXT, address TEXT, city TEXT, state TEXT, country TEXT DEFAULT 'India', pincode TEXT, start_date DATE, time_slot TEXT, duration TEXT, budget TEXT, special_skills TEXT, source TEXT, notes TEXT, admin_notes TEXT, status TEXT DEFAULT 'New', submitted_at TIMESTAMPTZ DEFAULT NOW(), updated_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS marketplace ( id BIGSERIAL PRIMARY KEY, ref_id TEXT, type TEXT, full_name TEXT, whatsapp TEXT, service TEXT, city TEXT, experience TEXT, rate TEXT, availability TEXT, description TEXT, documents TEXT, budget TEXT, urgency TEXT, admin_notes TEXT, status TEXT DEFAULT 'New', submitted_at TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE IF NOT EXISTS pageviews ( id BIGSERIAL PRIMARY KEY, page TEXT, referrer TEXT, is_new_session BOOLEAN DEFAULT true, visited_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE leads ENABLE ROW LEVEL SECURITY; ALTER TABLE marketplace ENABLE ROW LEVEL SECURITY; ALTER TABLE pageviews ENABLE ROW LEVEL SECURITY; CREATE POLICY IF NOT EXISTS "ins_leads" ON leads FOR INSERT WITH CHECK (true); CREATE POLICY IF NOT EXISTS "ins_mkt" ON marketplace FOR INSERT WITH CHECK (true); CREATE POLICY IF NOT EXISTS "ins_pv" ON pageviews FOR INSERT WITH CHECK (true); CREATE POLICY IF NOT EXISTS "all_leads" ON leads FOR ALL USING (true); CREATE POLICY IF NOT EXISTS "all_mkt" ON marketplace FOR ALL USING (true); CREATE POLICY IF NOT EXISTS "all_pv" ON pageviews FOR ALL USING (true);
🔔
